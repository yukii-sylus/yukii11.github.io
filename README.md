# Flask WebApp Demo

## 项目介绍
一个简单的注册+登录网站，使用 Flask + SQLite。

## 技术栈
- Flask
- SQLite
- HTML/CSS
- 云部署（腾讯云）

## 本地运行
```bash
pip install flask
python app.py
```

## 云部署步骤
- 腾讯云注册与服务器获取
- 使用 Git 克隆项目
- 设置 Python 环境
- 安装依赖并运行 Flask
- 设置 Nginx 代理
